Name Type Argument Default Description
xGrid boolean

yGrid boolean

whether draw the grid line started from Y axes
xStyle string <optional>
 'value'

the X axes's style,'value' or 'class'
yStyle string <optional>
 'value'

the Y axes's style,'value' or 'percentage%'
classes Array(string)

xDescript string <optional>
 'x'

yDescript string <optional>
 'y'

xSpan double

the increment on x value in each grid,(data space)
ySpan double

the increment on y value in each grid,(data space)
lineWidth double <optional>
 '1'

the lineWidth of text
